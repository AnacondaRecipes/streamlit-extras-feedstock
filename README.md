Installing streamlit-extras
==================

Installing `streamlit-extras` from the main channel can be achieved by:

```
conda install streamlit-extras
```

It is possible to list all of the versions of `streamlit-extras` available on your platform with `conda`:

```
conda search streamlit-extras
```
